## get start
#### 1. 使用该框架请设置CSS
```
*,*::before,*::after{
box-sizing:border-box
}
```
IE 8及以上支持此样式


IE 15及以上支持此样式
#### 2. 安装sovt
```shell script
  npm i -S sovt
```
#### 3. 引入
```js
import {Button,ButtonGroup,Icon} from 'sovt'
import 'sovt/dist/index.css'
export default {
  name: 'App',
  components: {
    'g-button':Button
  }
}
```
